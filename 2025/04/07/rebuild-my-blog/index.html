重建我的 Blog

经过一番折腾，我终于成功重建了我的博客！地址是 https://tmakerchima.github.io/blog2025/，这次使用的是 Hexo 和 GitHub Pages。虽然过程中遇到了不少问题，比如图片不显示、404 错误等，但最终都一一解决了。这篇文章记录了我的重建过程和心得。

为什么要重建？

我的旧博客因为配置混乱和部署问题，经常出现页面打不开的情况。于是我决定从头开始，重新搭建一个简洁、稳定的博客，用来记录学习和技术分享。

重建步骤

  1. 初始化 Hexo 项目
    • 我安装了 Hexo CLI，然后运行：
      hexo init blog2025
cd blog2025
npm install
